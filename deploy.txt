برای دیپلوی پروژه و کدها ابتدا لازم است که nodejs +14 روی سیستم شما نصب باشد.
در پوشه پروژه، ترمینال را باز کنید و دستورات زیر را به ترتیب اجرا کنید:

1- npm i
2- npm run build
3- npm run start

سپس همانظور که در ترمینال مشاهده می کنید، پروژه شما روی localhost:3000 اجرا شده است.
الان میتوانید با تنظیم وب سرور، پروژه خود را آنلاین کنید.
