این با پروژه Nextjs +15 App Router, Typescript, TailwindCSS ساخته شده است.
برای توسعه سریع تر و استفاده از کامپوننت های قابل استفاده مجدد و قابل شخصی سازی شدن، از Shadcn/ui استفاده شده است.
تمامی کد ها در پوشه src می باشند.
در پوشه app کد های مربوط به صفحات پروژه قرار گرفته اند.
در پوشه components کدهای مربوط به کامپوننت های استفاده شده در پروژه قرار دارند. همچنین پوشه ui که درون پوشه components قرار دارد، کدهای مربوط به صرفا ui قرار دارد. این کامپوننت ها در سراسر برای طراحی رابط کاربری پروژه استفاده شده اند.
در پوشه entities کدهای مروبط به تایپ دیتاهای استفاده شده در پروژه قرار دارند.
در پوشه hooks، هوک های کاسوم قرار دارند. برای مثال هوک use-mobile.tsx برای چک کردن اندازه صفحه می باشد(مثل مدیا کوئری css ) که با JS پیاده شده است.
در پوشه lib، کدهای مربوط به کتابخانه های درون پروژه ای قرار دارند.
در پوشه providers، برای استفاده از کتابخانه هایی مثل React-Query-Tanstak، نیاز است یک provider برای آن ساخت و در برنامه از آن استفاده کرد. این پوشه محل نگهداری این کد هاست.
پوشه schemas محل نگهداری shcemaهای فرم ها می باشد.
پوشه services مربوط به کدهای ارتباط با backend می باشد که ساختار api client در آن پیاده شده است.
و در نهایت پوشه utils که مربوط به کدها و فایل هایی است که نگهداری و خانایی بالاتر فانکشن های بهم مرتبط، آن ها را در این پوشه به صورت فایل مجزا نگهداری میکنیم.
